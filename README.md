# HR Management System

A full-stack HR management web application designed to manage employee records, roles, and internal workflows.

## Features
- Employee management (add, update, delete)
- Role-based access
- REST API backend
- MySQL database integration
- Secure authentication (in progress)

## Tech Stack
- Backend: Flask
- Frontend: Vue.js, HTML, CSS, JavaScript
- Database: MySQL
- APIs: REST API
- Tools: Git, GitHub
